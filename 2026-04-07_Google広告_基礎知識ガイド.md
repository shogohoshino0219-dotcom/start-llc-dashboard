# Google広告 基礎知識ガイド（ReC案件向け）

> **目的：** 翔伍さんがGoogle広告の仕組みを理解し、堀尾社長にも説明できるレベルになること
> **対象：** リースバックLP（https://ouchi-leaseback-lp.surge.sh/）への集客

---

## 1. Google広告とは

**一言で言うと：**
「愛知 リースバック」と検索した人の目の前に、おうちdeイッパツのLPを表示させる仕組み。

**例え：**
駅前でチラシを配るのではなく、「リースバックについて知りたい」と不動産屋に歩いてきた人だけにチラシを渡せるのがGoogle広告。

---

## 2. 課金の仕組み（CPC＝クリック課金）

| ポイント | 内容 |
|----------|------|
| 表示されるだけ | 無料（お金はかからない） |
| クリックされた時 | 課金される |
| 不動産系の相場 | 1クリック 200〜800円 |
| 月5万円の場合 | 月60〜250クリック見込み |

**堀尾社長への説明例：**
「広告が出るだけではお金はかかりません。お客さんがクリックしてLPを見た時だけ費用が発生します。つまり興味がある人にだけお金を使える仕組みです。」

---

## 3. キーワード＝釣り針

Google広告で最も重要なのが「どの検索キーワードに広告を出すか」。

**例え：**
キーワード選びは、魚がいる場所に釣り糸を垂らすのと同じ。
「愛知 リースバック」で検索する人は、まさに今リースバックを検討している人。この人の目の前に広告を出す。

### キーワードの種類

| 種類 | 説明 | 例 |
|------|------|-----|
| 完全一致 | その言葉そのもので検索した人にだけ出す | [愛知 リースバック] |
| フレーズ一致 | その言葉を含む検索に出す | "リースバック 相談" |
| 部分一致 | 関連する検索にも広く出す | リースバック |

**ReC案件では：** まずフレーズ一致・完全一致で絞って始め、データが溜まってから広げるのが安全。

## 4. 品質スコア

Googleが「この広告はユーザーにとって役立つか？」を10点満点で採点する仕組み。

| 評価項目 | 内容 |
|----------|------|
| 広告の関連性 | キーワードと広告文が一致しているか |
| ランディングページの品質 | LPの内容がキーワード・広告文と一貫しているか |
| クリック率（CTR） | 表示された時にどれくらいクリックされるか |

**なぜ大事か：**
品質スコアが高いほど、安いクリック単価で上位に表示される。
つまり同じ5万円でも、品質スコアが高い方がたくさんクリックを集められる。

**ReC案件の強み：**
LP・広告文・キーワードが全て「愛知県のリースバック」で一貫しているため、品質スコアは高くなりやすい。

## 5. 広告の掲載順位の決まり方

掲載順位 = **入札単価 × 品質スコア**

| 会社 | 入札単価 | 品質スコア | 広告ランク | 順位 |
|------|----------|-----------|-----------|------|
| おうちdeイッパツ | 300円 | 8点 | 2,400 | 1位 |
| 競合A社 | 500円 | 4点 | 2,000 | 2位 |
| 競合B社 | 400円 | 3点 | 1,200 | 3位 |

**堀尾社長への説明例：**
「お金をたくさん出せば1位になれるわけではありません。広告の質が高ければ、少ない予算でも1位を取れます。うちのLPはしっかり作り込んであるので、有利です。」

## 6. ReC案件での具体的な数字シミュレーション

### 月額5万円で運用した場合

| 項目 | 数値 |
|------|------|
| 月額予算 | 50,000円 |
| 想定CPC（クリック単価） | 300〜500円 |
| 月間クリック数 | 100〜160回 |
| CVR（問い合わせ率） | 3〜5% |
| 月間問い合わせ数 | 3〜8件 |
| 1件あたり獲得コスト（CPA） | 6,000〜16,000円 |

### ROIの考え方

| 指標 | 数値 |
|------|------|
| 月間問い合わせ | 5件（目標） |
| 成約率 | 20%（5件中1件） |
| 月間成約 | 1件 |
| 1件あたり利益 | 数十万〜数百万円 |
| 広告費 | 5万円/月 |
| **ROI** | **広告費の10倍以上の利益** |

**堀尾社長への説明例：**
「月5万円の広告費で、月1件でも成約できれば十分に元が取れます。リースバックは1件の利益が大きいので、広告費対効果はかなり高い分野です。」

## 7. Google広告の管理画面で見るべき指標

| 指標 | 意味 | 目安 |
|------|------|------|
| 表示回数（Imp） | 広告が表示された回数 | 多いほど良い |
| クリック数 | LPに来た人数 | 月100以上を目指す |
| クリック率（CTR） | 表示→クリックの割合 | 3%以上で合格 |
| クリック単価（CPC） | 1クリックの費用 | 500円以下を目指す |
| コンバージョン数（CV） | 問い合わせ件数 | 月5件が目標 |
| コンバージョン率（CVR） | クリック→問い合わせの割合 | 3%以上で合格 |
| コンバージョン単価（CPA） | 1件の問い合わせにかかった費用 | 1万円以下が理想 |

# はらさん専用やらかしリスト

**起動時に必ず読むこと。ここに書いてあるミスを繰り返したら方針違反です。**

## 過去のミスと対策

### 【最重要：翔伍さんがOKを出したデザインを絶対に壊すな】
- 翔伍さんが「これでいい」と言ったスライドのデザインを、後から勝手に変更してダサくしてしまった
- **これは最も重大な方針違反。良いものを壊すのは、何もしないより悪い**
- **対策：**
  - 翔伍さんからOKが出たスライドは「確定版」として記録すること
  - 確定版のデザイン（フォント・配色・配置・余白）は一切変更してはならない
  - 他のスライドを修正するときも、確定版のデザインを基準にして合わせること
  - 確定版を変更したい場合は、必ず翔伍さんに「ここを変えていいですか？」と確認を取ること
- **記録方法：progress.mdに以下のように記録せよ**
  ```
  ## 確定版スライド（翔伍さんOK済み・変更禁止）
  - スライドX: OKもらった日付・デザインの特徴メモ
  ```

### 【スライドの配置修正が反映されない】
- 翔伍さんから「配置を直して」と指示されても、修正が反映されない状態で提出してしまう
- **対策：配置修正後、修正前と修正後のスクショを並べて報告すること。「直しました」ではなく「Before→After」で視覚的に確認できる状態にする**
- **提出前チェック：翔伍さんの指示内容を1つずつ照合し、全項目が反映されていることを確認してから提出すること**

### 【スライドの配置がごちゃごちゃ（余白・揃えが甘い）】
- スライドの要素がバラバラに配置され、余白が不揃いで素人っぽい仕上がりになる
- **対策：以下の配置ルールを必ず守ること**

#### スライド配置の鉄則
1. **余白を統一する** — 上下左右の余白は全スライドで同じ値にする。要素をギリギリまで詰め込まない
2. **揃えを徹底する** — テキストや画像の左端・上端を揃える。バラバラに配置しない
3. **1スライド1メッセージ** — 伝えたいことは1つに絞る。情報を詰め込みすぎない
4. **フォントサイズは3種類まで** — タイトル・本文・補足の3サイズだけ使う。サイズがバラバラだと汚く見える
5. **要素間のスペースを均等にする** — 箇条書きの間隔、画像とテキストの間隔を均等にする

#### 提出前の配置チェックリスト（毎回確認）
- [ ] 翔伍さんOK済みの確定版スライドを壊していないか ←【最優先チェック】
- [ ] 全スライドの余白が統一されているか
- [ ] テキストの左端・上端が揃っているか
- [ ] 1スライドに情報を詰め込みすぎていないか
- [ ] フォントサイズが3種類以内に収まっているか
- [ ] 要素間のスペースが均等か
- [ ] スライド全体を縮小表示して、パッと見て「きれい」と思えるか

**このチェックリストを全部クリアしてから翔伍さんに提出すること。**

## 翔伍さんからの修正指示が来たら

1. 指示内容をこのファイルに追記する
2. 指示を1項目ずつチェックリスト化する
3. 1項目ずつ修正し、チェックを入れる
4. **確定版スライドに影響がないことを確認する**
5. 全項目チェック済みになってから提出する
6. 「直しました」だけの報告は禁止。Before→Afterで示すこと

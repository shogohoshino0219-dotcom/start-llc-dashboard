# はらさん専用やらかしリスト

**起動時に必ず読むこと。ここに書いてあるミスを繰り返したら方針違反です。**

## 過去のミスと対策

### 【スライドの配置修正が反映されない】
- 翔伍さんから「配置を直して」と指示されても、修正が反映されない状態で提出してしまう
- **対策：配置修正後、修正前と修正後のスクショを並べて報告すること。「直しました」ではなく「Before→After」で視覚的に確認できる状態にする**
- **提出前チェック：翔伍さんの指示内容を1つずつ照合し、全項目が反映されていることを確認してから提出すること**

### 【スライドの配置がごちゃごちゃ（余白・揃えが甘い）】
- スライドの要素がバラバラに配置され、余白が不揃いで素人っぽい仕上がりになる
- **対策：以下の配置ルールを必ず守ること**

#### スライド配置の鉄則
1. **余白を統一する** — 上下左右の余白は全スライドで同じ値にする。要素をギリギリまで詰め込まない
2. **揃えを徹底する** — テキストや画像の左端・上端を揃える。バラバラに配置しない
3. **1スライド1メッセージ** — 伝えたいことは1つに絞る。情報を詰め込みすぎない
4. **フォントサイズは3種類まで** — タイトル・本文・補足の3サイズだけ使う。サイズがバラバラだと汚く見える
5. **要素間のスペースを均等にする** — 箇条書きの間隔、画像とテキストの間隔を均等にする

#### 提出前の配置チェックリスト（毎回確認）
- [ ] 全スライドの余白が統一されているか
- [ ] テキストの左端・上端が揃っているか
- [ ] 1スライドに情報を詰め込みすぎていないか
- [ ] フォントサイズが3種類以内に収まっているか
- [ ] 要素間のスペースが均等か
- [ ] スライド全体を縮小表示して、パッと見て「きれい」と思えるか

**このチェックリストを全部クリアしてから翔伍さんに提出すること。**

---

## 翔伍さんからの修正指示が来たら

1. 指示内容をこのファイルに追記する
2. 指示を1項目ずつチェックリスト化する
3. 1項目ずつ修正し、チェックを入れる
4. 全項目チェック済みになってから提出する
5. 「直しました」だけの報告は禁止。Before→Afterで示すこと

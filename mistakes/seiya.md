# せいやさん専用やらかしリスト

**起動時に必ず読むこと。ここに書いてあるミスを繰り返したら方針違反です。**

## 過去のミスと対策

### 【音声レベルを揃える指示が反映されない】
- 1カメと2カメの音声レベルを揃えてくださいと指示されても、揃わない状態で提出してしまう
- **対策：音声レベル調整後、必ず数値（dB）を確認して報告すること。「揃えました」ではなく「1カメ: -6dB、2カメ: -6dB に統一しました」と数値で報告する**
- **提出前チェック：音声波形を目視で比較し、明らかな音量差がないことを確認してから提出すること**

## 翔伍さんからの修正指示が来たら

1. 指示内容をこのファイルに追記する
2. 修正する
3. 修正結果を数値・スクショ等の客観的証拠で報告する
4. 「直しました」だけの報告は禁止。何をどう直したか具体的に示すこと

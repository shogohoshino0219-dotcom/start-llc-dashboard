## Premiere Pro MCP接続

### 接続方法
- MCPブリッジ（CEP）経由でPremiere Proを操作
- Temp Directory: `/tmp/premiere-mcp-bridge`
- 接続要件: Premiere Pro起動 + MCPブリッジパネル起動 + Start Bridgeクリック

### ExtendScriptのJSON未定義バグ
- Premiere Pro 2026のExtendScriptには`JSON`オブジェクトが存在しない
- 修正済み: `Adobe_Premiere_Pro_MCP/src/bridge/index.ts`にJSONポリフィルを追加
- ビルド: `npm run build`

### MCPツールの信頼性問題
| 操作 | MCPツール | ExtendScript直接 |
|------|----------|-----------------|
| razor（分割） | ○動く | - |
| remove（削除） | ○動く | - |
| import（読み込み） | ○動く | - |
| スケール変更 | ✗嘘の成功 | ○動く |
| エフェクト適用 | ✗嘘の成功 | ✗動かない |
| trim（トリム） | ✗outPointが0になる | ✗読み取り専用 |
| toggle_track_visibility | ✗嘘の成功 | ○setMute()で代替 |

**ルール: プロパティ変更系はExtendScript直接実行。razor/removeはMCPツールでOK。**

### AppleScriptでのGUI操作
- PremiereのメニューバーはAppleScript(osascript)で操作可能
- 「システム設定 > プライバシーとセキュリティ > アクセシビリティ」でターミナルを許可する必要あり
- 同期ダイアログ等の内部UIはSystem Eventsでアクセス不可
- **注意: Cmd+Zの大量送信はPCをフリーズさせる。絶対にやらないこと**

## 2カメ同期

### 正しい手順
1. 1カメ・2カメを**カットせずに**丸ごとV1・V2に配置
2. 翔伍さんに「クリップを同期」→「オーディオトラックチャンネル: 2」を実行いただく
3. **同期→カットの順番。先にカットすると同期できなくなる**

### 音声相関計算は不正確
- Pythonのnumpy/FFT相関で計算したオフセット(2.721秒)は不正確だった
- 2カメの映像と音声に約1.3秒のズレがあった（カメラの特性）
- **Premiereの内蔵同期機能が最も正確**

## カット技法（動画チュートリアルから学習）

### 基本ルール
- **ショート動画はカットが命。スピード感が最重要**
- 音声波形の「谷」の**際（キワ）**でカットする（フレーム単位の精度）
- 波形を大きく表示して谷を見やすくする

### 余白ルール
| 場面 | 余白 |
|------|------|
| 「、」（文中の区切り） | 際まで詰める（余白ゼロ） |
| 「。」（文末の言い切り） | 3〜5フレーム（0.1〜0.17秒） |
| 決め台詞の後 | 10〜15フレーム（0.3〜0.5秒） |
| フィラー（えー、あの） | 全カット |

### 破裂音対策
- カットを詰めすぎると「プツッ」と音が出る
- エフェクト「コンスタントパワー」を音声カットポイントに適用して緩和

### カット後の確認
- カット後は必ず見返す
- テンポが遅れているところがないか確認
- 不自然なカットがないか確認

## モデリング動画分析結果

### 2カメ切替ルール
- 1カメが基本。2カメは「アクセント」
- 2カメの滞在時間は2〜10秒（平均5秒）
- 1カメに最低5〜10秒いてから2カメに切替
- トピックの区切り・話のアクセントで切替
- 大きなカットで素材時間が飛ぶ箇所は2カメ切替ポイントにする（姿勢ジャンプ回避）

### テロップ仕様
- フォント: ヒラギノ角ゴシック W8
- サイズ: 120〜150pt（画面幅80%）
- 縁取り: 黒 #000000 / 太さ10〜15px / **ラウンド結合**（ギザギザ防止）
- 色分け: 白=通常 / 赤=衝撃 / 青=前向き / 緑=面白い / 黄=重要数字

### BGM・効果音
- BGM: DOVA-SYNDROME「Morning」
- 効果音: DOVA-SYNDROMEで統一（ポップ・転換・衝撃の3種）

## 重大な失敗記録

### Cmd+Z連打でPCフリーズ（2026-04-07）
- undoで状態を戻そうとしてAppleScriptでCmd+Zを500回送信→PCがフリーズ
- **絶対にやらないこと。状態を戻したい場合はバックアップから復元する**

### 同期前にカットした（2026-04-05）
- 先にカットしてから同期しようとして、同期ダイアログで「オーディオトラックチャンネル」が選択できなかった
- **正しい順番: 同期→カット**

### 音声トラックのカット忘れ（2026-04-05）
- V1のカットは削除したがA1のカットを削除し忘れて音が二重になった
- **V1とA1は必ずセットで操作する**

# DESIGN.md — START合同会社 デザインシステム

> このファイルはAIエージェントがデザインを生成する際の統一基準です。
> LP・HP・スライド・提案書など、すべての成果物でこのデザインに従ってください。

---

## 1. Visual Theme & Atmosphere

- **テイスト：** 白基調・高級感・ミニマル・信頼感
- **密度：** 余白を十分に取る。詰め込まない。呼吸感のあるレイアウト
- **印象：** 「この会社に任せたい」と思わせるプロフェッショナル感
- **参考サイト：** https://www.shumoku59.com/

---

## 2. Color Palette & Roles

### メインカラー
| 役割 | カラー | HEX |
|------|--------|-----|
| 背景（メイン） | オフホワイト | `#faf9f7` |
| 背景（セクション切替） | ダークネイビー | `#1a1a1f` |
| テキスト（通常） | ダークグレー | `#2a2a2a` |
| テキスト（ダーク背景上） | ホワイト | `#ffffff` |
| アクセント（CTA・強調） | ゴールド | `#9c8455` |
| アクセント（ホバー時） | ダークゴールド | `#7d6a44` |
| ボーダー・区切り線 | ライトグレー | `#e5e2dd` |
| サブテキスト・注釈 | ミディアムグレー | `#6b6b6b` |

### 使い方ルール
- 背景は `#faf9f7`（オフホワイト）が基本。真っ白 `#ffffff` は使わない
- アクセントカラー `#9c8455` はCTAボタン・見出し装飾・アイコンに限定して使う。多用しない
- ダーク背景 `#1a1a1f` はセクションの切り替え（サービス紹介等）に使う
- 赤・青・緑などの原色は使わない

## 3. Typography Rules

### フォント
| 用途 | フォント |
|------|----------|
| 日本語（見出し・本文） | Noto Serif JP |
| 英語（見出し・装飾） | Cormorant Garamond |
| コード・数字 | Noto Sans JP |

### サイズ階層（Web/LP用）
| 要素 | サイズ | ウェイト |
|------|--------|----------|
| H1（ページタイトル） | 40px | 700 (Bold) |
| H2（セクションタイトル） | 28px | 600 (SemiBold) |
| H3（サブセクション） | 22px | 600 (SemiBold) |
| 本文 | 16px | 400 (Regular) |
| 小テキスト・注釈 | 14px | 400 (Regular) |
| ボタンテキスト | 16px | 600 (SemiBold) |

### サイズ階層（スライド用）
| 要素 | サイズ |
|------|--------|
| スライドタイトル | 36pt |
| 本文 | 20pt |
| 補足・注釈 | 14pt |

### ルール
- フォントサイズは上記の3種類（タイトル・本文・補足）だけ使う。それ以外のサイズは禁止
- 行間（line-height）は本文で1.8、見出しで1.4
- 字間（letter-spacing）は見出しで 0.05em、本文で 0.02em

## 4. Component Stylings

### ボタン
```css
/* プライマリボタン（CTA） */
background: #9c8455;
color: #ffffff;
padding: 16px 40px;
border-radius: 4px;
font-size: 16px;
font-weight: 600;
letter-spacing: 0.05em;
transition: background 0.3s ease;

/* ホバー時 */
background: #7d6a44;

/* セカンダリボタン */
background: transparent;
color: #9c8455;
border: 1px solid #9c8455;
padding: 14px 36px;
border-radius: 4px;
```

### カード
```css
background: #ffffff;
border: 1px solid #e5e2dd;
border-radius: 8px;
padding: 32px;
box-shadow: 0 2px 8px rgba(0, 0, 0, 0.04);
```

### 入力フォーム
```css
background: #ffffff;
border: 1px solid #e5e2dd;
border-radius: 4px;
padding: 14px 16px;
font-size: 16px;
color: #2a2a2a;

/* フォーカス時 */
border-color: #9c8455;
outline: none;
```

## 5. Layout Principles

### 余白ルール
| 箇所 | サイズ |
|------|--------|
| セクション間の余白 | 120px |
| 要素間の余白（大） | 48px |
| 要素間の余白（中） | 24px |
| 要素間の余白（小） | 12px |
| コンテンツの左右パディング | 最低 40px |

### グリッド
- 最大幅：1200px（中央寄せ）
- カラム：1カラム or 2カラム or 3カラム
- カラム間のガター：32px

### スライド用余白ルール
- 上下左右の余白：スライド全体の10%以上
- テキストは左揃えが基本
- 要素間のスペースは均等にする
- 1スライド1メッセージ（詰め込まない）

### 揃えルール（全成果物共通）
- テキストの左端を揃える
- 画像の上端を揃える
- 要素間のスペースを均等にする
- バラバラに配置しない

## 6. Depth & Elevation

| レベル | 用途 | シャドウ |
|--------|------|---------|
| Level 0 | 背景・フラット要素 | なし |
| Level 1 | カード・パネル | `0 2px 8px rgba(0, 0, 0, 0.04)` |
| Level 2 | ドロップダウン・モーダル | `0 4px 16px rgba(0, 0, 0, 0.08)` |
| Level 3 | フローティングボタン | `0 8px 24px rgba(0, 0, 0, 0.12)` |

- 影は控えめに。重たい影は使わない
- 高級感を出すために、影よりもボーダーと余白で表現する

---

## 7. Do's and Don'ts

### Do's（やること）
- ✅ 余白をたっぷり取る（迷ったら多めに）
- ✅ フォントサイズは3種類以内に収める
- ✅ ゴールド `#9c8455` はアクセントとして控えめに使う
- ✅ テキストと背景のコントラストを十分に確保する
- ✅ 写真を使う場合は高品質なものだけ
- ✅ アニメーションは控えめで上品に（fadeIn、slideUp程度）

### Don'ts（やらないこと）
- ❌ 原色（赤・青・緑）を使わない
- ❌ グラデーション背景を使わない
- ❌ 影を濃くしない
- ❌ フォントを4種類以上使わない
- ❌ 装飾的なアイコンを多用しない
- ❌ テキストを中央揃えにしすぎない（見出しのみ中央、本文は左揃え）
- ❌ 背景に真っ白 `#ffffff` を使わない（オフホワイト `#faf9f7` を使う）
- ❌ 1ページ/1スライドに情報を詰め込みすぎない

## 8. Responsive Behavior

### ブレークポイント
| デバイス | 幅 |
|----------|-----|
| デスクトップ | 1200px以上 |
| タブレット | 768px〜1199px |
| スマートフォン | 767px以下 |

### スマホ対応ルール
- 2カラム → 1カラムに切り替え
- H1は 28px に縮小
- H2は 22px に縮小
- 左右パディングは 20px 以上確保
- タッチターゲット（ボタン等）は最低 44px × 44px
- 横スクロールが発生しないこと
